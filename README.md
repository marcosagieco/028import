*028import 💨 - Catálogo Digital y E-commerce*

Desarrollé esta plataforma para digitalizar el catálogo de productos de una tienda de vapes y automatizar el proceso de pedidos, que anteriormente dependía exclusivamente de canales manuales. Mi objetivo fue crear una experiencia de usuario fluida y un sistema centralizado para el negocio.

Decisiones Técnicas y Funcionalidades:
Desarrollo con Next.js: Decidí utilizar Next.js para aprovechar el renderizado del lado del servidor (SSR) y las rutas dinámicas, garantizando un rendimiento superior y un posicionamiento eficiente.

Gestión de Datos y Stock: Implementé Firebase (Firestore) para manejar el catálogo en tiempo real. Esto permite que cualquier modificación en el inventario se refleje de inmediato para el cliente final.

Experiencia de Usuario (UX): Utilicé Tailwind CSS para construir una interfaz responsiva y liviana, priorizando la navegación desde dispositivos móviles, que representa el mayor flujo de usuarios de la tienda.

Administración Centralizada: Diseñé un panel administrativo privado para que el dueño pueda gestionar productos y visualizar pedidos sin necesidad de manipular el código directamente.

Stack Tecnológico:
Framework: Next.js

Base de Datos: Firebase (Firestore)

Autenticación: Firebase Auth

Estilos: Tailwind CSS

Hosting: Vercel
Este proyecto nació para digitalizar y organizar las ventas de 028import, una tienda de vapes. El objetivo fue centralizar el catálogo y automatizar el proceso de pedidos que antes se manejaba de forma manual por WhatsApp.

🛠️ Lo que más destaco: El Admin Dashboard
Es la parte de la que más orgulloso estoy. Desarrollé un panel administrativo privado donde el dueño tiene control total sin tocar código:

Gestión de contenido: Editar, agregar o pausar productos en tiempo real.

Control de Stock: Cambios instantáneos en el catálogo del cliente.

Gestión de Pedidos: Centralización de datos para despachos rápidos.

🏗️ Tech Stack
Elegí estas herramientas para lograr un sitio ultra liviano y rápido:

Next.js: Rendimiento superior y ruteo limpio.

Firebase (Firestore & Auth): Base de datos en tiempo real y seguridad en el panel admin.

Tailwind CSS: Diseño responsive enfocado en dispositivos móviles.

Vercel: Despliegue automático y hosting.
